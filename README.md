# Kubota Corporation (kubota-corporation)

Kubota Corporation is a Japanese multinational manufacturing agricultural, construction, and industrial machinery, including compact tractors, excavators, mowers, utility vehicles, engines, and water and environmental infrastructure systems sold in over 120 countries. Kubota does not currently publish a public developer API or developer portal; equipment telematics (KubotaNOW) and dealer integrations are handled through partner channels rather than open APIs.
